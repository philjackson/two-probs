# re

Simple clojure regexp library.

## Usage

### Simple chars

* `\W` (`:W`, `:non-word`)
* `\A` (`:A`, `:beginning-input`, `:beg-input`, `:beg-in`)
* `\b` (`:b`, `:boundary`, `:bndry`)
* `\B` (`:B`, `:non-boundary`, `:non-bndry`)
* `\w` (`:w`, `:word`)
* `\Z` (`:Z`, `:end-term`)
* `\S` (`:S`, `:non-space`)
* `\z` (`:z`, `:end-input`, `:end-in`)
* `\D` (`:D`, `:non-digit`)
* `.` (`:.`, `:any`)
* `\d` (`:d`, `:digit`)
* `\s` (`:s`, `:space`)
* `^` (`:start`, `:beg`, `:beginning`)
* `$` (`:$`, `:end`)
* `\G` (`:G`, `:end-match`)
